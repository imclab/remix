h1. Everything is a Remix

A simple animated version of "Kirby Ferguson":http://www.kirbyferguson.com/ 's graphic for explaining the processes of remixing.

"See it live here":http://vallandingham.me/remix/

Remix to your hearts content.

Built with D3.js

h2. Usage

h3. First Clone the Repository

<pre>
git clone git@github.com:vlandham/remix.git
cd remix
</pre>

You can start up a local webserver by using python's simple server:

<pre>
python -m SimpleHTTPServer -p 3000
</pre>

or @thin@ in ruby

<pre>
thin start
</pre>

Now go to "http://0.0.0.0:3000":http://0.0.0.0:3000/

and you have a basic visualization
